## 🌆 What is NeuroFleetX?

**NeuroFleetX** is a next-generation urban mobility intelligence platform that combines **AI, real-time data, and interactive visualization** to help cities and organizations manage fleets and traffic more efficiently.

Built with scalability and modern architectures in mind, NeuroFleetX reflects **industry-grade development practices** suitable for real-world deployment.

## 🧱 Tech Stack

### 🎨 Frontend

* Angular 19 **or** React 19
* Component-based, responsive UI

### ⚙️ Backend

* Java Spring Boot
* RESTful APIs

### 🗄️ Database

* MySQL
* Structured, relational data storage

### 🧠 AI & Intelligence

* Python Flask Microservice **or** Java-based ML
* Traffic analytics, prediction, and decision logic

### 🗺️ Maps & Location

* Google Maps API
* Leaflet.js
* Real-time route and vehicle visualization

### 🔐 Security

* Spring Security
* JWT-based authentication & authorization

### 📈 Data Visualization

* Chart.js
* ApexCharts
* Recharts

### 🔌 Integrations & Services

* SMTP Email Service
* WebSockets for live updates
